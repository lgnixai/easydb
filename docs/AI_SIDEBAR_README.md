# AI 侧边栏 - 总览

> 🤖 通过自然语言管理数据库，效率提升 80%+

## 🎯 一句话介绍

将原本空白无用的右侧边栏改造成强大的 AI 助手，通过 Ollama 模型和 MCP 实现自然语言操作数据库。

## ✨ 核心亮点

- 🚀 **效率翻倍**: 从 10 步操作变 1 步
- 💬 **自然交互**: 像聊天一样简单
- 🎯 **智能理解**: 自动识别上下文
- 🔒 **本地部署**: 数据不出本地
- 🎨 **完美集成**: 与现有 UI 风格统一

## ❓ 常见问题

### Q: 必须安装 Ollama 吗？

A: 是的，目前只支持 Ollama。未来可能支持 OpenAI、Claude 等。

### Q: 数据安全吗？

A: 非常安全！AI 运行在本地，数据不会发送到外部服务器。

### Q: 支持哪些语言？

A: 目前优化了中文支持，英文也可以使用。

### Q: 可以自定义吗？

A: 可以！修改配置文件即可调整模型、温度等参数。
