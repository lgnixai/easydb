# AI 侧边栏演示脚本

## 演示目标

展示 AI 侧边栏如何通过自然语言简化数据库操作，提升 80% 以上的效率。

## 演示准备

### 1. 环境准备（演示前完成）

```bash
# 启动 Ollama
ollama serve

# 确认模型已下载
ollama list | grep llama3.2

# 启动项目
cd teable-ui
pnpm dev
```

### 2. 浏览器准备

- 打开 http://localhost:5173
- 登录系统
- 确保右侧边栏可见
- 准备好演示数据

### 第 1 幕: 介绍（1 分钟）

**[展示界面]**

**[指向 AI 侧边栏]**

> "这个 AI 助手可以通过自然语言帮我们管理数据库、表格、字段等，让我演示一下它有多强大。"

### 第 2 幕: 创建表格（2 分钟）

**[传统方式对比]**

> "传统方式创建一个表格需要这样..."

**[快速演示传统方式]**
1. 点击"创建表格"按钮
2. 填写表单
3. 点击确定
4. 再点击"添加字段"
5. 选择类型...

> "至少需要 10 次点击！现在看 AI 怎么做..."

**[AI 方式]**

在 AI 侧边栏输入：
```
创建一个员工信息表
```

> "就这么简单！AI 理解了我的意图，询问是否确认。"

**[点击确认]**

> "✅ 创建成功！左边已经出现了新表。一句话搞定！"

### 第 3 幕: 批量创建字段（2 分钟）

**[输入命令]**
```
给员工信息表添加以下字段：
- 姓名（文本，必填）
- 邮箱（邮箱，必填，唯一）
- 电话（电话）
- 部门（单选）
- 入职日期（日期）
- 是否在职（复选框）
```

> "看，一次性创建 6 个字段，还能指定类型和属性！"

**[AI 响应]**

> "AI 解析出了所有参数，询问确认..."

**[点击确认]**

> "✅ 全部创建成功！传统方式这需要至少 30 次点击，现在只需要 2 次！"

### 第 4 幕: 查询操作（1 分钟）

**[输入命令]**

```
列出当前数据库的所有表格
```

> "AI 还能帮我们查询信息..."

**[显示结果]**

> "立即得到了所有表格列表，不用手动翻找了。"

### 第 5 幕: 上下文理解（2 分钟）

## 互动 Q&A

**A**: 
- Ollama（本地 AI 服务）
- 支持的模型（推荐 llama3.2）
- 浏览器

### Q2: 安全吗？

**A**:
- ✅ 本地运行，数据不外传
- ✅ 重要操作需要确认
- ✅ 完整的错误处理

### Q3: 支持哪些操作？

**A**:
- 创建空间、数据库、表格
- 创建字段（8 种类型）
- 创建记录
- 查询信息

### Q4: 如何开始使用？

**A**:
```bash
# 1. 安装 Ollama
brew install ollama

# 2. 下载模型
ollama pull llama3.2

# 3. 启动项目
pnpm dev
```
